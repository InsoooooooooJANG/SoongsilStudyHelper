# SoongsilStudyHelper
숭실대학교 학습도우미 - 공지사항과 온라인 강의의 진행도, 과제에 대한 알림을 줍니다.

### 서비스 화면

![service_capture](./image/숭실대학습도우미.png)

### 기능
#### 1차 
* 공지사항과 수강하고 있는 수업을 크롤링해와서 한 페이지에서 보여준다.
#### 2차 (예정)
* 특정 키워드에 대한 알림 기능
* 공지사항 바로가기 링크
* 수업 바로가기 링크

### 사용한 기술
* java
* Spring boot
* Selenium
